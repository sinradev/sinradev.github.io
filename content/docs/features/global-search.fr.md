## Déclencher la Recherche

| Méthode | Raccourci |
|---------|-----------|
| Recherche modale | `Maj + F` |
| Recherche inline | `/` (met le focus sur le champ de recherche) |

Les deux ouvrent la même expérience de recherche. La modal apparaît au-dessus de n'importe quelle page sans perdre votre contexte actuel.

## Ce qui est Recherché

La recherche globale interroge cinq types d'entités simultanément :

| Type | Champs recherchés |
|------|------------------|
| **Issues** | Titre, contenu du corps |
| **Capabilities** | Nom, contenu du corps |
| **Projets** | Nom, description |
| **Releases** | Nom, numéro de version |
| **Pages** | Nom, contenu du corps |

Les résultats sont groupés par type, vous pouvez donc scanner chaque catégorie indépendamment.

## Éléments Récemment Consultés

Quand vous ouvrez la recherche avec une requête vide, Sinra affiche vos éléments récemment consultés au lieu d'un état vide. Cela rend la recherche globale utile comme outil de navigation, pas seulement comme outil de recherche.

Commencez à taper pour remplacer les éléments récemment consultés par des résultats de recherche en direct.

## Navigation dans les Résultats

Naviguez dans les résultats avec le clavier :
- `↑` / `↓` : se déplacer entre les résultats
- `Entrée` : ouvrir le résultat sélectionné
- `Échap` : fermer la recherche sans naviguer

## Périmètre

Les résultats de recherche sont limités à votre organisation actuelle. Vous ne verrez jamais de résultats d'une autre organisation même si vous êtes membre de plusieurs organisations.

Les résultats sont également filtrés par votre niveau d'autorisation. Les issues dans des projets auxquels vous n'avez pas accès n'apparaîtront pas.
